# 木

**木日主生春天**

木生春月旺，水多主灾殃。

不是贫困守，就是少年亡。

最喜戊己土，见火大吉昌。

见官反无官，庚辛命不强。

> 解析：春季木旺，但仍残存寒气，所以水多不吉，喜戊己土克水，或者火来抵御寒气。

**木日主生夏天**

夏木不为凋，火炎怕土燥，

火盛要水养，水盛木则漂。

最喜见寅卯，子辰亦为好，

财官若根轻，午未运不高。

> 解析：夏季木休

> 水火平衡：
> 
> 水是救星（制火），但忌过量（水多木漂）。
> 
> 扎根第一：
> 
> 地支见寅卯（木根）、子辰（水润）为最佳配置。
> 
> 警惕燥土：
> 
> 未戌土是“抽水机”，午火是“放大镜”，避之则吉。
> 
> 财官看根基：
> 
> 土金（财官）虚浮时，午未运易引发破财失业。

**木日主生四季月**

木生四季月，怎把戊己克，

但求壬癸水，反宜金叠叠，

丑辰须用火，未戌水木扶，

全是火土现，土运亦为安。

> 解析:
> 
> 绝不硬刚：土旺时木别妄想克土，优先求 水（生木） + 金（生水泄土）。
> 
> 因地施策：
> 
> 丑辰寒土 → 用 火 暖局（火是催化剂）。
> 
> 未戌燥土 → 用 水木 扎根（水木是生存基础）。
> 
> 极端选择：火土强到极致时，直接“投降”从财（火土），反得富贵。
> 
> 

**木日主生秋月**

秋木逢旺金，金旺及伤身，

不宜重见土，干头爱丙丁。

金木若相战，财运反遭陷，

壬癸若相助，定然名利亨。

> 解析：
> 
> 头号救星：丙丁火
> → 克金护身 + 暖局解冻（无火必凶）。
> 
> 二号救星：壬癸水（需配合火）
> → 泄金生木 + 润局养根（杀印相生贵格）。
> 
> 大忌：
> 
> 土：生金克木，埋木毁根；
> 
> 金木硬刚：无火水通关则人财两损。

**木日主生冬月**

冬木枯叶调，水冷怕金削。

有土相逢火，温暖发根苗。

冬天水木泛，名利总虚无。

运逢火土助，名利才自如。

> 解析：
> 
> 首需火土：
> 
> 火是命根（无火必败），
> 
> 土是基石（制水培木）。
> 
> 大忌金水：
> 
> 金生水更寒（雪上加霜），
> 
> 水多木漂（根基全无）。
> 
> 水木泛滥必改运：
> 
> 原局水木多者，需等火土运（中年后）才能发迹。

# 火

**火日主生春天**

火到春见阳，木盛火必强。

水旺金不旺，富贵理应当。

火多木必焚，最喜见官乡。

支中见多卯，反倒为灾殃。

> 解析：火命春生，木火要适度，太旺需水来调

**火日主生夏天**

夏火非寻常，木多主不祥。

水旺要金相，方才显高强。

火盛土必燥，无水焉能好。

宜壬不宜癸，不须寅卯绕。

> 解析：夏季火旺，木可助燃，有水方为水火相济,喜水;金为火之财星，但夏季金为最微弱的死势，金强，可使财运趋于旺盛之势，喜金



**火日主生四季月**

火生四季衰，水多常有害。

用金不和谐，甲己喜相爱。

未戌多孤寡，丑辰是良才。

休逢戊己旺，必定有祸殃。

> 解析：火在四季末（辰、戌、丑、未月）最弱。忌水金：水克火，金耗火，二者加速火灭。喜：甲木（生火）+己土（制水）

> 未、戌：燥土（夏季余火），吸干火力，火弱遇之更枯竭 → 孤寡（精力不济、人缘淡薄）。

> 丑、辰：湿土（含水气），能培木生火，且土制水护火 → 良才（根基稳、得助力）。

**火日主生秋天**

秋月火不强，唯喜甲丙帮。

用木还用土，土重则为殃。

岁运逢火地，自然财兴旺。

水盛最无情，不夭眼目盲。

> 解析：秋季火属囚势，好比黄昏，喜木，得木生火可助其势;秋水为相，土可制水，也喜土，也土过多，则土多火晦;岁运逢火，益增光辉，能克金生财

**火日主生冬天**

火生冬不旺，四柱怕金乡。

爱用甲乙木，干适戊己乡。

人生若逢此，必定是官郎。

只恐财杀旺，刑冲见阎王。

> 解析：火在冬天死绝之气，喜欢用木生火，用土克水。不喜金（生水耗火）+ 水（杀）无制。


# 土

**土日主生春天**

春月生戊己，难抵甲和乙。

用金必有利，有火方是奇。

不愁春水旺，身衰反无依。

水火两相济，方能有名利。

> 救急靠金：
> → 砍伐压制你的木（小人/压力），但需火防金过锋（伤土气）。
> 
> 扎根靠火：
> → 火是命根（生土抗木），无火则土虚。
> 
> 发财靠水平衡：
> → 水是财，但必须土强（有火生）才能接住，否则财变祸。

**土日主生夏天**

戊己夏月生，干头喜见金。

旺遇火丙丁，无水不显荣。

水火名既济，戊己得安生。

但愁木火土，焦躁身不宁。

> 救命核心：金水
> → 金泄土生水（挖井引泉），水制火润燥（降温保湿）。
> 
> 格局关键：水火既济
> → 水火互控成平衡，土命方得安生富贵。
> 
> 大忌组合：木火土
> → 木生火→火生土→土燥金水绝，必致焦枯败亡。

**土日主生四季土月**

四季戊己旺，不宜火焰乡。

甲乙喜相逢，用水定吉昌。

丑辰单用火，卯旺名声扬。

戌未水木乡，戊己忌再帮。

> 通病：土过旺 → 首需 木疏土、水润局（致富关键）。
> 
> 因地施策：
> 
> 寒土（丑辰）：补小火暖局 → 忌大火（防燥）。
> 
> 燥土（未戌）：补水木扎根 → 忌再见土（防埋金水）。
> 
> 虚名陷阱：
> 
> 土无木水 → 看似强壮（身旺），实则贫瘠（无财官）；
> 
> 土得卯制 → 反因自律（官星）成名。

**土日主生秋天**

秋月逢戊己，金旺泻土气。

用火方成器，用水不相宜。

此时愁木旺，怎能无灾殃。

或是遭人欺，或是有残疾。

> 唯一救星：火
> → 制金（防泄气） + 生土（补元气） → 无火必败。
> 
> 两大死敌：
> 
> 金：过度消耗你（泄土）；
> 
> 木：趁虚打压你（克土）；
> 
> 水：落井下石（耗土助寒）。
> 
> 极端避讳：木金联手
> → 金泄土虚 + 木克土崩 → 灾殃残疾之兆。

**土日主生冬季**

戊己逢冬生，见金则不荣。

甲乙虽为病，有火是公卿。

支中若无根，为人定漂流。

须得戊己帮，亥子是财乡。

> 最佳配置：
> 土（堤坝） + 火（能源） → 化洪水（水）为资源（富贵）
> 
> 大忌组合：
> 金（生水助洪） + 水（泛滥） → 冲毁火（财）土（业）
> 
> 自立根基：
> 
> 火土弱则早年漂泊（祖业难靠）；
> 
> 火土运至时中年发迹（扎根自立）。

# 水

春水不当权，木盛夺水源。

用金礼相见，见火乐封源。

水木伤官格，喜见财与官。

水木两相见，空来人世间。

**水日主生夏天**

水生夏月天，木多火炎炎。

休言财官旺，只要不枯源。

金水若相涵，名利两相欢。

土燥金不见，不能成方圆。

> → 唯一水源（无金必败），且需远离火克（如地支藏申酉护金）。

> 格局核心：金水相涵

> 致命雷区：
> 
> 木多：助火耗水（自焚）；

> 土燥无金：克水绝源（事业崩盘）；

> 火旺无制：耗干元气（身心衰竭）。
